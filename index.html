HEIGHT-BALANCED KD-TREES

The KD-tree is a binary tree representation of multi-dimensional data structures like quad-trees. This project involved implementing a height-balanced variant of the standard KD-tree in Java. The rebalancing is done by destroying the subtree with the height imbalance, and then rebuilding it.

  • Insert operation inserts a k-dimensional point into the tree and rebalances if necessary.
  • Delete operation deletes a KD point and rebalances if necessary.
  • Search operation will look for a KD point and return it if it exists in the tree. Otherwise it will throw an exception.
  • Nearest neighbor query: Given a point p that necessarily may not exist in the tree, this query uses the Pythagorean distance formula to find the closest point to p and return that point.
  • Range query: Given a KD bounding box, this query returns all the points which are located inside the box.
